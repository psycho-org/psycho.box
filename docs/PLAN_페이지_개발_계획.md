# PLAN: 페이지 개발 계획

메뉴별로 개발할 페이지 목록과 우선순위입니다.  
참고: PlanIt UI (다크 테마, 보라색 액센트)

---

## 1. 인증 (Auth Layout)

| 페이지 | 경로 | 우선순위 | 비고 |
|--------|------|----------|------|
| 로그인 | `/login` | P0 | 이메일/비밀번호, OAuth(선택) |
| 회원가입 | `/register` | P0 | OTP 플로우 (requests → confirmations → register) |
| 비밀번호 찾기 | `/forgot-password` | P1 | OTP 요청/확인 후 재설정 |

## 2. 내 워크스페이스 (App Layout)

| 페이지 | 경로 | 우선순위 | 비고 |
|--------|------|----------|------|
| 내 워크스페이스 | `/workspaces` | P0 | 통계 카드, 워크스페이스 카드 목록, + 워크스페이스 생성 |
| 워크스페이스 상세(리다이렉트) | `/workspaces/[id]` | P0 | `/workspaces/[id]/board`로 리다이렉트 |

**내 워크스페이스 페이지 구성**

- 통계 카드: 참여 워크스페이스 수, 활성 프로젝트 수, 완료된 태스크 수, 팀원 수
- 워크스페이스 카드: 아바타, 이름, 역할(오너/멤버), 프로젝트/태스크/멤버 수, 멤버 아바타, 활동 상태
- + 새 워크스페이스 생성 카드

---

## 3. 스프린트 보드 (Workspace Layout)

| 페이지 | 경로 | 우선순위 | 비고 |
|--------|------|----------|------|
| 스프린트 보드 | `/workspaces/[id]/board` | P0 | Kanban (할일/진행중/검토중/완료) |
| 스프린트 상세(접기/펼치기) | 동일 페이지 내 | P0 | 진행률 바, 스프린트 완료 버튼 |

**스프린트 보드 페이지 구성**

- 컨텍스트: 프로젝트명, 날짜 범위
- 액션: + 스프린트 생성, 내보내기
- 필터: 전체, 내 태스크, 높은 우선순위, 마감 임박
- Kanban 컬럼: 할일, 진행중, 검토중, 완료
- 태스크 카드: 타입(Feature/Design/Bug), 제목, ID, 담당자, 우선순위
- 컬럼 하단: + 태스크 추가

---

## 4. 태스크 (Workspace Layout)

| 페이지 | 경로 | 우선순위 | 비고 |
|--------|------|----------|------|
| 모든 태스크 | `/workspaces/[id]/tasks` | P0 | 카드/리스트 뷰 전환, 필터·보기 옵션 |

**모든 태스크 페이지 구성**

- **보기 타입 (View Type):**
  - **카드형**: 그리드/카드 레이아웃. 태스크를 카드 형태로 표시 (타입, 제목, ID, 담당자, 우선순위, 상태 등)
  - **리스트형**: 테이블/리스트 레이아웃. ID, 태스크, 담당자, 스프린트, 우선순위, 상태 컬럼
- **필터·보기 옵션 (Filter):**
  - **전체**: 모든 태스크
  - **내 태스크**: 담당자가 나인 태스크만
  - **백로그**: 스프린트에 할당되지 않은 태스크
  - **로드맵**: 스프린트·프로젝트 기반 타임라인/로드맵 뷰
  - **완료됨**: 완료된 태스크만
- **보기 타입 전환**: 카드/리스트 토글 (아이콘 또는 탭)

## 5. 멤버 관리 (Workspace Layout)

| 페이지 | 경로 | 우선순위 | 비고 |
|--------|------|----------|------|
| 멤버 관리 | `/workspaces/[id]/members` | P0 | 이메일 초대, 멤버 목록 |

**멤버 관리 페이지 구성**

- 초대 폼: 이메일, 역할(멤버), 초대 보내기
- 멤버 목록: 아바타, 이름, 이메일, 역할(오너/멤버), 설정/취소 버튼
- 초대 대기중 표시

## 6. 모달 (공통)

| 모달 | 트리거 | 우선순위 | 비고 |
|------|--------|----------|------|
| 새 태스크 생성 | + 태스크 생성 버튼 | P0 | 제목*, 설명, 스프린트, 우선순위, 담당자, 태그 |
| 스프린트 생성 | + 스프린트 생성 버튼 | P0 | 이름*, 목표, 시작일*, 종료일* |
| 워크스페이스 생성 | + 워크스페이스 생성 | P0 | 이름*, 설명, 아이콘 색상 |
| 멤버 초대 | + 멤버 초대 버튼 | P0 | 이메일*, 역할 |
| 태스크 상세/편집 | 태스크 카드 클릭 | P0 | 설명, 댓글, 상태, 우선순위, 담당자, 스프린트, 태그 |

## 10. 에러·안내 (App Layout)

| 페이지 | 경로 | 우선순위 | 비고 |
|--------|------|----------|------|
| 워크스페이스 접근 권한 없음 | `/access-denied` | P0 | 403 시 리다이렉트. "이 워크스페이스에 대한 접근 권한이 없습니다", 워크스페이스 목록으로 이동 버튼 |

**접근 권한 없음 페이지 구성**

- 메시지: "이 워크스페이스에 대한 접근 권한이 없습니다"
- 설명: 탈퇴했거나 삭제된 워크스페이스일 수 있습니다
- 버튼: "워크스페이스 목록으로" → `/workspaces` 이동

## 11. 우선순위 요약

### P0 (MVP)

- 로그인, 회원가입
- 워크스페이스 접근 권한 없음 (`/access-denied`)
- 내 워크스페이스 (통계, 카드, 생성)
- 스프린트 보드 (Kanban)
- 모든 태스크 (카드/리스트 뷰, 백로그/로드맵/내 태스크 등 필터)
- 멤버 관리 (초대, 목록)
- 모달: 태스크 생성, 스프린트 생성, 워크스페이스 생성, 멤버 초대, 태스크 상세/편집

### P1

- 비밀번호 찾기
- 워크스페이스 설정
- 소유권 이전
- 계정 설정
- AI 분석 대시 (API 연동 가능 시)

### P2

- AI 분석 상세
- 회원 탈퇴
